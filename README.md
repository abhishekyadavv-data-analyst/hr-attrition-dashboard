# HR Attrition Dashboard

## Project Overview
Interactive Excel Dashboard built on IBM HR 
Analytics Employee Attrition dataset.

## Dataset
- Source: Kaggle (IBM HR Analytics Dataset)
- Rows: 1,470 employees
- Features: 35 columns

## Features
- Dynamic Slicers (Department, Gender, JobRole)
- Dynamic KPI Cards
- 5 Interactive Charts

## KPI Cards
- Total Employees: 1,470
- Attrition Count: 237
- Attrition Rate: 16.1%
- Avg Monthly Income: 6,502
- Active Employees: 1,233

## Charts
- Attrition Overview (Pie Chart)
- Department Wise Attrition (Bar Chart)
- Gender Wise Attrition (Donut Chart)
- Age Group Wise Attrition (Bar Chart)
- Job Role Wise Attrition (Bar Chart)

## Key Findings
- Attrition Rate: 16.1%
- Sales department highest attrition
- Age group 18-27 most likely to leave
- Male attrition higher than Female

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Dynamic Formulas
